Important FAQ:

What is the Document Object Model (DOM)?

The DOM (Document Object Model) is a programming interface representing an HTML document as a structured tree, allowing JavaScript to interact with and manipulate elements.

How do you select an element from the DOM?

To select elements in the DOM, use methods like getElementById, getElementsByClassName, or querySelector to find specific elements by ID, class, or CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is when an event listener is attached to a parent element to handle events from its child elements. It improves performance by reducing multiple listeners and can manage dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

We can manipulate attributes using setAttribute() or getAttribute() and change an element's styles directly through the style property, like element.style.color.
